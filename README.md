# Mega Mart Admin Dashboard

Next.js 16.1.6 admin dashboard for managing products, vendors, and orders.

## 🚀 Quick Start

```bash
cd admin
npm install
npm run dev    # Development
npm run build  # Production build
npm start      # Run production
```

## 📋 Features

✅ Product management  
✅ Category and brand management  
✅ Vendor approval system  
✅ Order management  
✅ User management  
✅ Sales analytics and dashboard  
✅ System settings
